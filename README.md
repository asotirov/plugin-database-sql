# plugin-database-sql

# Usage

> npm install --save git+https://github.com/asotirov/plugin-database-sql.git

## Include in serverConfig.js

>     module.exports = [
>     ....
>     {
>        packagePath: '../node_modules/plugin-database-sql'
>     }

## package.json:

>       ..
>       "plugin": {
>          "consumes": [
>             "sql"
>          ],
>           "provides": [
>             "database-sql"
